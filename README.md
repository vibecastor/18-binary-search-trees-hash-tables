18: Binary Search Trees and Hash Tables
===

### Binary Search Class
 - remove(node) method
    - Big O 
        - time: O(log(n)) 
        - space: O(H) where height is the height of the tree

### K-ary Tree Class
 - find(value)
    - Big O
        - time: O(n^2) 
        - space: O(H) where H is the height of the tree
 
 - toString()
    - Big O
        - time: O(n^2)
        - space: O(H) where H is the height of the tree
 
 - toArray(array) 
    - Big O
         - time: O(n^2)
         - space: O(H) where H is the height of the tree
